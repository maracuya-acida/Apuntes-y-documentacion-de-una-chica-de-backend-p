# ¿Que es un hosting?ᓚᘏᗢ
##### Es el conjunto de servicios que se encarga de llevar las paginas webs a otros dispositivos.

## Algunos tipos de hosting
### - Hosting compartido: es un tipo de alojamiento web y se comparten la potencia y la ram del servidor-
Los archivos no se comparten pero se mantienen en la misma parte fisica

### - *Hosting VPS*: Este tipo de alojamiento tienen un espacio unico para la web.
Debes tener un conocimiento tecnico para el hosting de este tipo de servidores
### - *Hosting Cloud*: Este tipo de alojamiento se parece el vps, pero existe una instancia aislada la gestion
las hacen los provedores de alojamiento. (se le recomienda a tiendas con un gran grosor de clientes)
### - Hosting dedicado: Es una maquina fisica dentro de un centro de datos, se debe tener un conocimiento altamente tecnico,
es una opcion muy flexible. Se recomienda para paginas webs muy potentes.
#### Plugins de provedores de hosting para wordpress
- Siteground (limitaciones en algunos paises y las renovaciones son costosas)
- Bluehost (tiene un panel intuitivo pero es costoso)
- Go daddy (cumple pero tiene planes muy caros, pero tiene soporte 24 horas)
- Hostgator (es igual que bluehost)
- WB engine (tiene soporte especializado, tiene una gran velocidad y sirve para tiendas grandes )
- Hostinguer (es confiable y economico, ofrece un dominio rapido aparte que tiene buenas herramientas y no tiene soporte tecnico)
